Starting KPIs

Initial values for all tracked indicators at the start of the mission. These are the baseline the player works from in Week 1.

These values directly affect difficulty. Low starting budget + low supply = immediate pressure. High security + high acceptance = easier first weeks.

Step 4 of 5

Events

Define the events that can fire during the mission. Each event has a title, type, description, 3 choices, and 3 outcome levels per choice (immediate, medium-term, long-term).

Each event needs at least 3 choices. Each choice needs 3 outcome descriptions and optional KPI effects. Dice weights (success / partial / fail) must sum to 100.
Step 5 of 5

Playable Positions

Which positions can the player choose for this scenario? Each position gets starting stat modifiers and a unique opening briefing.

Only include positions that are actually on this mission (from your Staffing section). A position can only be playable if it's Tier 1 in staffing.
